Methods

Animation:new (path, width, height, framesDuration)
Creates an animation.

Parameters:

  • path the patch to the spritesheet.
  • width the width of the sprite.
  • height the height of the sprite.
  • framesDuration a table with ordered numeric keys from 1 to the number of sprites. Each key must have the duration of its sprite, in frames.
