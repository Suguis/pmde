Methods

Dungeon:new (name, totalFloors, width, height, tilesetId)
Creates a new dungeon

Parameters:

  • name string the name of the Dungeon.
  • totalFloors number the number of floors.
  • width number the width of each Floor in the dungeon, in cells.
  • height number the height of each Floor in the dungeon, in cells.
  • tilesetId string the name of the tileset.

Returns:

    Dungeon the new dungeon.
Dungeon:getFloor (floor)
Draws a floor of the dungeon.

Parameters:

  • floor number the number of the floor to draw (absolute value).

Raises:

an error if the floor doesn't exist in the dungeon.
