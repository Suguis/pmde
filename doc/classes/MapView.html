MapView:new (pos, floor, tileset)
Creates a new MapView.

Parameters:

  • pos Vector the vector of the coordinates that the MapView is going to put in the center.
  • floor Floor the Floor on which the MapView is going to focus.
  • tileset Image the Image that the MapView buffer is going to use.

Returns:

    MapView the new MapView.
