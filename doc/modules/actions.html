Functions

keys.getAction (key)
Gets an action from a key and returns it.

Parameters:

  • key string the key from which calculate the action.

Returns:

    string the action associated to that key.
keys.getMoveVector (moveAction)
Returns a Vector from a movement action.

Parameters:

  • moveAction string the action that represents a direction.

Returns:

    Vector the unit Vector of the direction, or nil if the action is not a move action.
