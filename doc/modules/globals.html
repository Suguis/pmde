globals
A table containing own defined global variables

Fields:

  • CELL_SIZE the size of the cells, in pixels.
  • TILESET_WIDTH the width of the tilesets images.
  • TILESET_HEIGHT the width of the tilesets images.
  • TILES_DISPLAY_WIDTH The number of tile columns that can be displayed.
  • TILES_DISPLAY_HEIGHT The number of tile rows that can be displayed.
  • time the time passed in seconds since the beggining of the game.
  • frames the frames passed since the beggining of the game.
